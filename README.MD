Installing Python environment
For courses using Python you will need Python and Python Extension for VS Code to run your source code.
Please install Python following the instructions below.

Installing Python
Go to https://www.python.org/downloads/
Download the latest version of Python 3.
Double-click the installation file to start the installation.
Make sure you choose "Add Python 3.x to PATH" (see picture point 1).
Choose "Install now" (see picture point 2).
Once installation is completed, you can close the install window.
